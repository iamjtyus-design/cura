# CURA Design System

## Direction

CURA should feel premium, restrained, clear, and native.

The design should communicate practical intelligence, not generic AI spectacle.

## Typography

### App

Use Apple system typography for readability, Dynamic Type, accessibility, and native behavior.

### Marketing

Use Poppins for the VisionBuilt wordmark and selected marketing headlines.

## Core Colors

Directional tokens:

```text
ink: #10151D
deep: #172334
canvas: #F7F5F0
white: #FFFFFF
signal: #316BFF
signalDark: #1E4FCC
gold: #B98A46
softGold: #E7D4AE
success: #1F8A5B
warning: #B7791F
danger: #C94A4A
muted: #667085
border: #D9DEE7
```

Validate every combination for accessibility before release.

## Semantic States

1. Draft.
2. Capturing.
3. Uploading.
4. Processing.
5. Partial.
6. Complete.
7. Warning.
8. Failed.
9. Private.
10. Cloud.

Status must never depend on color alone.

## Spacing

Use a consistent 4-point base.

Suggested scale:

```text
4, 8, 12, 16, 20, 24, 32, 40, 48
```

## Shape

1. Small radius for controls.
2. Medium radius for cards.
3. Large radius for sheets and prominent panels.
4. Avoid excessive pill shapes.

## Key Components

1. Capture Action Card.
2. Session Card.
3. Source Card.
4. Processing Stage Row.
5. Curated Note Section.
6. Output Card.
7. Evidence Chip.
8. Inference Badge.
9. Quick Send Button.
10. Export Sheet.
11. Mode Selector.
12. Privacy Mode Selector.
13. Folder Row.
14. Empty State.
15. Visual Brief Preview.

## Capture Button

The Capture action should be unmistakable, reachable, and accessible.

It should not resemble a hidden recording control.

## Motion

Motion should communicate state:

1. Recording started.
2. Recording paused.
3. Processing stage changed.
4. Output completed.
5. Copy succeeded.
6. Export succeeded.

Avoid confetti, excessive bounce, and decorative animation.

## Icon Direction

Use simple line icons.

Avoid:

1. Robot.
2. Brain.
3. Sparkles as the main identity.
4. Generic microphone as the app logo.
5. Cartoon mascot.

## CURA Mark

Use a strong C-based symbol that can suggest capture, clarity, curation, and forward movement.

## Accessibility

1. Dynamic Type.
2. VoiceOver.
3. Minimum touch targets.
4. Reduce Motion.
5. High contrast.
6. Visible focus.
7. Non-color status cues.
8. Accessible charts and Visual Briefs.
